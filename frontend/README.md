# Frontend AquaIA (PMV1)

## Vistas mínimas PMV1
- `/login`
- `/` dashboard + mapa Leaflet
- `/alerts`
- `/zones`
- `/sensors`
- `/users`

## Ejecutar
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Variables
- `NEXT_PUBLIC_API_URL=http://localhost:8000/api/v1`


> Para instalación completa con backend y PostgreSQL, revisa `../docs/guia-instalacion.md`.
